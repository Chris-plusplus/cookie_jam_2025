# Catsino

In *Catsino*, you take on the role of a gambling cat determined to win as many tasty treats as possible.

Just like in a real casino, you can win big… or lose even more.

And yet, one thought always comes back:
**“Maybe… just one more round?”**

---

## Important

To run the game, you must install:

* **Visual C++ Redistributable Packages 2022** (link provided below)
* The included custom font

---

## About the Project

*Catsino* is the first game developed using the **Archimedes game engine**, created by the **AGH Code Industry student research group**.

---

## System Requirements

* x64 architecture processor
* Dedicated graphics card with **Vulkan support**

---

## Installation

1. Install **Visual C++ Redistributable 2022**

   * Download here:
     https://learn.microsoft.com/pl-pl/cpp/windows/latest-supported-vc-redist?view=msvc-170
   * Choose the **x64** version

2. Install the font:

   * File: `PixelatedEleganceRegular-ovyAA.ttf`
   * Located in the same folder as this file

3. Run the game:

   * Navigate to the `run` folder
   * Launch `main.exe`

---

## Notes

* It is **not recommended to resize the game window**, especially switching to fullscreen mode
